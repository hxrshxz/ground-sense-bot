About This Demo

This is a demo website showing how to embed the Ground Sense Bot chatbot.

🚀 How to Embed

Simply add this script tag to your website:

<script src="https://ground-sense-bot.vercel.app/embed.js" data-api-key="your-gemini-api-key-here" data-position="bottom-right" data-theme="dark" data-color="#00d4ff"> </script>

Note: Replace your-gemini-api-key-here with your actual Gemini API key.
